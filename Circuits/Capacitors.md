Capacitors: an element that gradually stores charge through buildup
$$i_C(t) = C \frac{dv_C(t)}{dt}; v_c(t) = v_0 + \frac{1}{C}\int_{t_0}^{t}i_c(x)dx; W_C = \frac{1}{2}Cv_c^2(t)$$

- Steady: capacitors block all current from passing through but let all voltage through
- Charging: $V_C(t) = V_0 (1-e^\frac{-t}{RC})$ and 
- Discharging: $V_C(t) = V_0 e^\frac{-t}{RC}$
- $I(t) = \frac{V_0}{R}e^\frac{-t}{RC}$ when $t \neq 0$
### Capacitors in Series
$$C_{eq} = \frac{1}{\sum \frac{1}{C_n}} = \frac{L_1*L_2}{L_1 + L_2}$$
*$C_{eq}$ = equivalent capacitance, $C_n$ = capacitance per capacitor, $C_1 / C_2$ = capacitance of capacitor 1 & 2*

### Capacitors in Parallel
$$C_{eq} = \sum C_n$$
*$C_{eq}$ = equivalent capacitance, $C_n$ = capacitance per capacitor*
