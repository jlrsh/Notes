Resistors: element that resist current flow

### Resistors in Parallel
$$R_{eq} = \frac{1}{\sum \frac{1}{R_n}} = \frac{R_1*R_2}{R_1 + R_2}$$
*$R_{eq}$ = equivalent resistance, $R_n$ = resistance per resistor, $R_1 / R_2$ = resistors of resistor 1/2*

**Current Division:** given multiple resistors in parallel, current through one resistor can be found from sum of total resistance in loop
$$i_x = \frac{\frac{1}{R_x}}{\sum \frac{1}{R_n}} = i_{total} * \frac{R_1}{R_1 + R_2}$$
*$i_x$ = current through resistor with $R_x$*

### Resistors in Series
$$R_{eq} = \sum R_n$$
*$R_{eq}$ = equivalent resistance, $R_n$ = resistance per resistor*

**Voltage Division:** given multiple resistors in parallel, voltage through one resistor can be found from sum of total resistance in loop
$$i_x = i_{total} * \frac{\frac{1}{R_x}}{\sum \frac{1}{R_n}}$$
*$i_x$ = current through resistor with $R_x$
