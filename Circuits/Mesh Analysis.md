Mesh Analysis: applying [[Kirchoff's Voltage Law|KVL]] to all loops in a circuit to find all loop [[Sources|current source]]

- When finding loop currents, edge conditions must be accounted for between loops
- IE if loop 1 and loop 2 share a border, $I_1 - I_2$ must be somewhere in the equation
- when solving, solve algebraically and then solve for actual values later
