Resistance: measure of difficulty of current flow through an element

### General Approach to Finding Equivalent [[Resistance|Resistance]]
1. Turn off all [[Sources|independent sources]] by shorting voltage sources and gapping current sources
2. If sources are dependent, apply a 1V or 1A test source to find dependent source characteristics
3. Combine all possible resistors using equivalents for series/parallel [[Resistors|Resistors]]
4. Solve circuit using [[Kirchoff's Voltage Law|KVL]] or [[Kirchoff's Current Law|KCL]], if using a test source $R_{eq} = \frac{V_t}{I_t}$
